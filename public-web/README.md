# Elite Bridge Public Webpage

This is an original Elite Bridge landing page inspired by care marketplace patterns without copying Care.com design, layout, or wording.

## What it includes

- Public homepage for Massachusetts home care support
- Family/agency call to action: request care
- Caregiver call to action: find care work
- Care service categories
- How it works for families/agencies and caregivers
- Trust and transparency section
- Example caregiver profile cards

## How to preview

Open `public-web/index.html` in a browser.

## Recommended deployment

Deploy this folder as a static site on Vercel, Netlify, GitHub Pages, or your existing Elite Bridge website hosting.
